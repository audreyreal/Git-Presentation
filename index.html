Git

What is Git?

Git is the most popular version control system, it was made in 2005 by Linus Torvalds, the creator of Linux. With Git you can make new changes, branch, merge and compare past versions of your repository on Git websites such as GitHub or GitLab.

Why use Git?

People use git because it allows developers to work on the same project simultaneously, as well as keep track of changes between different versions and have the ability to roll back to previous versions should something go wrong. The convenience of all that in one tool is unparalelled.

How to use Git?

To push code up to GitHub you first need to start a new repository. Open up a terminal, go to the project folder and start a git repository by using the command git init. Use the command git add . the you can 'commit' it by using git commit -m "Your message about the commit". After you have your files commited you should then create a new repository in GitHub copy the link to your repository then go back to the terminal and do git remote add origin (link from GitHub). Finally do git push origin master.
